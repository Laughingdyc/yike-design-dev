#### Avatar Props

| 参数          | 描述     | 类型            | 默认值       |
| ------------- | -------- | --------------- | ------------ |
| `image`       | 图片地址 | ` string`       | `'false'`    |
| `imageStyle`  | 图片样式 | `CSSProperties` | -            |
| `description` | 描述     | `string`        | `'暂无数据'` |
